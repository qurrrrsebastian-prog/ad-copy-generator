# Personal Finance Tracker

![Python](https://img.shields.io/badge/Python-3.14-blue)
![Flask](https://img.shields.io/badge/Flask-3.1-green)

## 📌 Deskripsi
Full-stack web app untuk tracking pengeluaran & pemasukan pribadi. Flask + SQLite + Jinja2 HTML. Data persists di database lokal.

## 🎯 Fitur
- Tambah transaksi (income/expense) dengan kategori
- Dashboard: total income, expense, balance
- Transaction history dengan filter
- Form validation & error handling
- Responsive minimal UI

## 🛠️ Tech Stack
- Python, Flask, SQLite, Jinja2, HTML/CSS

## 🚀 Cara Menjalankan

```bash
pip install -r requirements.txt
python app.py
```

Buka browser: http://localhost:5000

## 📊 Key Insight
- Tracking expense harian mengurangi pengeluaran impulsif 20-30%
- Kategori terbesar biasanya "Makan" & "Transport" untuk mahasiswa
- SQLite cukup untuk personal finance < 10,000 transaksi
