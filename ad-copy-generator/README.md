# Ad Copy Generator — AIDA Framework

![Python](https://img.shields.io/badge/Python-3.14-blue)
![Streamlit](https://img.shields.io/badge/Streamlit-1.58-red)

## 📌 Deskripsi
Generator copy iklan dengan framework AIDA untuk jasa B2B Indonesia (rope access, glass cleaning, building maintenance). Output: 3-5 variasi copy siap pakai untuk Google Ads & social media.

## 🎯 Fitur
- Input: produk, layanan, benefit, tone, kontak
- Output: 4 komponen AIDA (Attention, Interest, Desire, Action)
- 3-5 variasi per generate
- Copy-to-textarea untuk easy copy-paste
- Template-based (no API key required)

## 🛠️ Tech Stack
- Python, Streamlit

## 🚀 Cara Menjalankan

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Key Insight
- AIDA framework meningkatkan CTR 20-30% vs copy generic
- 3 variasi copy = A/B test ready untuk Google Ads
- Template Indonesia-specific untuk jasa B2B property
